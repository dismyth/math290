MATH 290

Category Theory and Type Theory
Mo/We 1:30-2:45
Location TBA

Description

Type theories are certain kinds of formal deduction systems that can be used instead of set theory to provide a ‘foundation for mathematics.’ Historically, type theories have been more studied in computer science, linguistics, and philosophy departments than in math departments, but this has been changing recently with the recognition that the algebraic structures arising in type theory bear an intimate relationship to algebraic structures arising in pure math (e.g. in algebraic topology and geomety).

This course will attempt to give an elementary introduction to type theory with minimal prerequisites. My plan is to follow the textbook ”Type Theory and Formal Proof” by Nederpelt and Geuvers, supplementing with other sources as necessary. A certain amount of category theory is indispensible for understanding type theory, so this will be introduced on an as-needed basis. A good reference for category theory is “An introduction to category theory” by Harold Simmons.
